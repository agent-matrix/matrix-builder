# Agent-Matrix Integration

Scout acts as the trend-intelligence layer for Agent-Matrix.

Flow:

```text
Scout detects opportunity
→ Matrix AI plans
→ Matrix Guardian validates trust and risk
→ Matrix Treasury checks cost/value
→ Matrix Architect builds artifacts
→ Matrix Hub stores reusable knowledge
```

Use:

```text
GET /api/v1/matrix/opportunities
```

or MCP tool:

```text
get_matrix_opportunities
```
